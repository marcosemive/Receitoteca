  O projeto apresentado visa a integração entre cozinheiros e entusiastas da gastronomia, por meio de uma aplicação web em que os usuários cadastrados como cozinheiros podem publicar suas receitas (com título, ingredientes, instruções, tempo de execução e porções), enquanto os usuários comuns podem visualizar, comentar e favoritar as receitas disponibilizadas.
